# Portfolio Redesign

Independent redesign prototype for Nan Yao's portfolio.

- Repository: https://github.com/Starry-nn/portfolio-redesign
- Expected GitHub Pages URL after Pages is enabled: https://starry-nn.github.io/portfolio-redesign/
- Source file: `index.html`

Design notes:

- Single static HTML file with no build system.
- System font stack only to avoid mixed-font rendering issues.
- Bilingual English / Chinese content toggle.
- Uses the existing public portrait and CV hosted from the main portfolio site.
